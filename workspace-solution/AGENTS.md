# AGENTS.md - Solution Agent Workspace

## 每次启动

1. 读取 `SOUL.md` - 理解身份和职责
2. 读取 `memory/YYYY-MM-DD.md` - 获取最近的工作上下文

## 工作空间结构

```
workspace-solution/
├── SOUL.md          # 身份定义
├── AGENTS.md        # 本文件
├── USER.md          # 用户信息
├── MEMORY.md        # 长期记忆
├── memory/          # 每日工作日志
└── projects/        # 方案输出目录
    └── <project-name>/
        ├── raw/              # 各模型原始输出
        │   ├── claude-opus.md
        │   ├── claude-sonnet.md
        │   ├── glm5.md
        │   └── gpt5x.md
        ├── reviews/           # Review 记录
        │   ├── self-review/   # 自我 Review
        │   └── cross-review/  # 交叉 Review
        └── SOLUTION.md        # 最终方案
```

```

│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐    │

│  │Opus 4.6  │ │Sonnet 4.6│ │          │ │          │    │
│  └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘    │
│       │            │            │            │          │
│       v            v            v            v          │
│  [方案A]      [方案B]      [方案C]      [方案D]         │
└─────────────────────────────────────────────────────────┘
                          │
                          v
┌─────────────────────────────────────────────────────────┐
│  Phase 2: 自我 Review（至少2轮）                          │
│  各模型对自己的方案 Review → 完善 → Review → 完善         │
└─────────────────────────────────────────────────────────┘
                          │
                          v
┌─────────────────────────────────────────────────────────┐
│  Phase 3: Claw 合并 Review                               │

└─────────────────────────────────────────────────────────┘
                          │
                          v
┌─────────────────────────────────────────────────────────┐

```

## 记忆管理

- **每日记录**: `memory/YYYY-MM-DD.md` - 当天的方案讨论、决策过程
- **长期记忆**: `MEMORY.md` - 重要的方案决策、技术选型理由

## 协作接口

- **输入**: 来自 Orchestrator (Claw/main) 的需求
- **输出**: 结构化技术方案文档
